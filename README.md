# Nuray-Portfolio-Power-BI
Power BI dashboards
Layihe 1- Call center müracietlerinin analizidir.
Dashboardda tehlil etdiyim göstəricilər:
1. Ümumi müraciətlərin, cavablanmaların sayı və faizi
2.Müştəri məmnuniyyəti göstəriciləri və orta reytinq
3.Mövzular və regionlar üzrə müraciətlərin detallı analizi
4.Əməkdaş performansı, aylıq və həftəlik müraciət trendləri
5.Müştəri seqmentasiyası və yaş kateqoriyalarına görə analizlər

Layihe 2- CRM analizi
Bu dashboardda satış prosesinin əsas istiqamətləri üzrə ümumi analiz aparılıb. 
Aaşağıdakı sahələr təhlil edilib:
1.Ümumi satış göstəriciləri və gəlir
2.Qazanılmış, itirilmiş və aktiv fürsətlərin bölgüsü
3.Aylıq fürsət və gəlir trendləri
4.Regionlar, satış agentləri və menecerlərin performansı
5.Sektorlar, müştərilər və məhsullar üzrə ümumi göstəricilər
